There are currently six mailing lists:

  1. board@huntingtonhillsinc.org
    This list is used to communicate to the current Huntington Hills board. If you send an email from the Contact page to the board, this is the email that is used. You may also use it directly. Emails send to this address need to be approved to reduce the amount of spam that is sent to board members. All non-spam emails are always approved.
  2. center@huntingtonhillsinc.org
    This list is only for those residents that live on Center Entrance.
  3. notices@huntingtonhillsinc.org
    This list is used by the board to send official messages (like notice of the annual meeting for example). Only board members can send messages to this list.
  4. residents@huntingtonhillsinc.org
    This list is meant to be used by Huntington Hills residents to communicate with each other. You can only send a message from your subscribed email (the email you receive list messages).
  5. spring_valley@huntingtonhillsinc.org
    This list is only for those residents that live on Spring Valley Drive.
  6. website-updates@huntingtonhillsinc.org
    Announcements about changes or updates to the Huntington Hills website are communicated with this list. Only the webmaster can send messages to this list.
