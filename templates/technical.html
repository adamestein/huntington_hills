Author

The Huntington Hills website was created and is maintained by Adam Stein (adam@huntingtonhillsinc.org).

Tools

Nvu was originally used to create the web pages. Nvu is a complete Web Authoring System for Linux desktop users as well as Microsoft Windows and Macintosh users to rival programs like FrontPage and Dreamweaver.

Nvu was replaced by Kompozer. Kompozer picks up where Nvu stopped (as Nvu is no longer supported).

Kompozer has been replaced by Bluefish due to Kompozer no longer being supported. Bluefish is a powerful editor targeted towards programmers and webdevelopers, with many options to write websites, scripts and programming code.

Django is a Python web framework. It is used to support pages that display dynamic data.

Web Design Templates

Three different templates were used in the design of this site. All templates came from Open Source Web Design.

Forested

The Forested design is a simple theme that invokes an old 1920's feel.

Photogenic

The Photogenic design is a simple theme for an image gallery.

Tree Hugger

The Tree Hugger design was tested on a Windows machine in Firefox 1.5, Internet Explorer 6.0, Netscape 8.1, Opera 7.54 and should hold its own in a text browser.

It's using valid XHTML 1.0 (strict) and CSS with a WCAG-A rating.

This design is released under the creative commons license, so do what you want with it and have fun!
